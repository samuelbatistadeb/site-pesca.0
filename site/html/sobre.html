Este projeto foi desenvolvido na tentativa de criar um site que apresenta uma rede social, no qual o principal tema é a pesca. Buscamos criar uma plataforma onde seria possível a um pescador encontrar os principais locais de pesca, além de poder registrar seus itens e suas pescas.

Desenvolvedores contribuintes: Sophia Bergameli, Yasmyn Carrilho e Samuel Batista.
